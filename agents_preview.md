# CosmicWeave 智能体系统界面与功能预览

## 双重价值智能体系统

CosmicWeave的智能体系统实现了双重价值方向，既为项目所有者提供"超级员工"，也为网站用户提供个性化体验。

### 模式切换功能

```
+--------------------------------------+
|                                      |
|  [用户模式 ⇄ 管理员模式] (右上角)     |
|                                      |
+--------------------------------------+
```

**管理员模式**：您作为项目所有者可以使用智能体作为"超级员工"，执行专业任务
**用户模式**：网站访问者可以体验智能体提供的个性化功能与服务

## 6. 系统实现细节

### 技术架构
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **存储**: 本地存储 (localStorage) 保存用户偏好和智能体状态
- **交互**: 事件驱动架构，模块化设计

### 核心文件结构
- `dual_agents.html`: 主界面HTML结构
- `agents_data.js`: 8大智能体数据定义
- `dual_mode.js`: 双重模式切换功能
- `dual_agents_init.js`: 智能体系统初始化与核心功能

### 关键实现功能
1. **双重模式系统**: 根据当前模式(管理员/用户)动态调整界面和功能
2. **智能体激活机制**: 通过toggleAgent()函数实现智能体的激活与停用
3. **工作流程展示**: 可视化展示每个智能体的专业工作流程
4. **任务管理系统**: 创建、分配、跟踪和管理智能体任务
5. **本地存储**: 保存用户偏好和系统状态，实现持久化体验

### 关键代码片段
```javascript
// 智能体激活函数
function toggleAgent(agentId) {
    const index = activeAgents.indexOf(agentId);
    
    if (index === -1) {
        // 激活智能体
        activeAgents.push(agentId);
        showNotification(`已激活 ${getAgentName(agentId)}`, 'success');
    } else {
        // 停用智能体
        activeAgents.splice(index, 1);
        showNotification(`已停用 ${getAgentName(agentId)}`, 'info');
    }
    
    // 更新UI
    renderAgentCards();
    updateTasksList();
    updateAdminDashboard();
    
    // 保存到本地存储
    saveToLocalStorage();
}

// 模式切换函数
function toggleMode() {
    currentMode = currentMode === 'admin' ? 'user' : 'admin';
    
    // 保存到本地存储
    localStorage.setItem('agentSystemMode', currentMode);
    
    // 更新按钮状态
    const modeSwitchBtn = document.getElementById('modeSwitchBtn');
    if (modeSwitchBtn) {
        modeSwitchBtn.className = `btn ${currentMode === 'admin' ? 'btn-highlight' : 'btn-outline'}`;
        modeSwitchBtn.innerHTML = `
            <i class="fas fa-user-shield"></i> 
            <span>${currentMode === 'admin' ? '管理员模式' : '用户模式'}</span>
        `;
    }
    
    // 更新UI
    updateUIForCurrentMode();
}
